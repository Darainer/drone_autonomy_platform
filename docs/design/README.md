# Design Documents

Feature/change designs, one per file: `DES-<NNN>-<slug>.md` (sequential,
never reused). Written before implementation via the `design` skill
(`.claude/skills/design/SKILL.md`), which contains the template and rules.

Every design references requirement UIDs from `docs/requirements/*.sdoc` and
links a test plan in `docs/test_plans/` before it can be marked Implemented.
